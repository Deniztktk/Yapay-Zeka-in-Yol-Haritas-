# 🧑‍💼 AI Rolleri ve Görevler

2026 itibarıyla AI/ML sektörü, **3.2:1 talep-arz oranı** ile yazılım dünyasının en fazla iş imkânı sunan alanı haline gelmiştir.

# 🔬 Data Scientist (Veri Bilimcisi)

**Kariyer Rotası:**

Data Analyst → Junior Data Scientist → Data Scientist → Senior → Lead Data Scientist

💵 Maaş Aralığı : (2026, ABD)** | $121K – $196K/yıl |
📍 Ana Görevler:  Büyük veri setlerini analiz etmek, istatistiksel modeller kurmak, tahmine dayalı modeller geliştirmek, hipotez testleri yapmak.
🔑 Temel Araçlar: Python, R, SQL, Jupyter, Pandas, Scikit-learn, Tableau |
🏫 Eğitim Lisans: (İstatistik, Matematik, BilgisayarMüh.), Yüksek Lisans tercih edilen |
🌟 Fark Yaratan Beceriler: İstatistiksel akıl yürütme, A/B test tasarımı, domain uzmanlığı, hikâye anlatımı |

- ML Engineer'dan Farkı: Data Scientist araştırma ve analiz odaklıdır; prototipler üretir. ML Engineer bu prototipleri production'a taşır.

# ⚙️ Machine Learning Engineer (ML Mühendisi)

**Kariyer Rotası:**

ML Engineer/Data Scientist → Senior Engineer → Solutions Architect → Principal Architect → Technical Fellow

💵 Maaş Aralığı: (2026, ABD) | $149K – $220K/yıl (Senior: $260K+) |
📍 Ana Görevler: ML modelleri tasarlamak, eğitmek ve production'a deploy etmek; ölçeklenebilir pipeline'lar inşa etmek; API entegrasyonu; model performansını izlemek |
🔑 Temel Araçlar: PyTorch, TensorFlow, Kubernetes, Docker, MLflow, Python |
🏫 Eğitim: Lisans CS/Matematik; Yüksek Lisans ideal; PhD araştırma rolleri için |
🌟 Fark Yaratan Beceriler: Yazılım mühendisliği temelleri, dağıtık sistemler, cloud altyapısı |

- Data Scientist'ten Farkı: MLE üretim sistemlerine odaklanır; yazılım mühendisliği altyapısına hakimdir. DS, keşif ve hipotez odaklıdır.
